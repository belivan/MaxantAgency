## Twitter Best Practices

### Timing:
- **Best times to post**: 8-10am EST, 12-1pm EST, 5-6pm EST (weekdays)
- **Avoid**: Late nights, weekends (unless targeting specific timezone)

### Engagement Tactics:
1. **Ask questions** in final tweet to drive replies
2. **Quote tweet yourself** 2 hours after posting to bump it
3. **Reply to every comment** in first hour
4. **Pin to profile** if it's getting traction
5. **Repost with new hook** after 48 hours if underperforming

### Visual Content:
- Threads with images get 3x more engagement
- Use screenshots, not stock photos
- Keep text overlays minimal
- Aspect ratio: 16:9 for best display

### Hashtags:
- Use 1-2 max (Twitter algorithm doesn't favor them like Instagram)
- Only if highly relevant: #buildinpublic #SaaS #webdesign

### Thread Structure:
- Hook (tweet 1) should be scroll-stopping
- Tweets 2-9: Deliver value
- Tweet 10: Soft CTA (not salesy)
- Optimal length: 8-12 tweets
