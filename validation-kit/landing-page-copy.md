## General Landing Page Best Practices

### Above the Fold
- Headline answers: "What is this?"
- Subheadline answers: "Why should I care?"
- CTA visible without scrolling
- Hero image shows the product

### Trust Signals
- Customer logos (if available)
- Testimonials with photos + company names
- Security badges (SSL, GDPR, SOC 2)
- Stats (users, reviews, results)

### Clear CTA Hierarchy
- **Primary CTA:** "Start Free Trial" (action-oriented, benefit-focused)
- **Secondary CTA:** "See Demo" or "View Sample" (lower commitment)
- Repeat CTA 3-4 times down the page

### Objection Handling
- FAQ section addresses common concerns
- Money-back guarantee reduces risk
- "No credit card required" for trials
- Social proof counters skepticism

### Mobile Optimization
- 60%+ traffic will be mobile
- Test on actual devices
- CTA buttons easy to tap (44√ó44px minimum)
- Text readable without zooming

### Conversion Optimization
- Use contrasting color for CTA buttons
- Add urgency ("14-day trial", "Limited spots")
- Social proof near CTA ("Join 500+ agencies")
- Remove navigation (reduce exits)

### A/B Testing Ideas
Test these elements:
- Headline variations
- CTA text ("Start Free Trial" vs "Get Started" vs "Try It Free")
- Hero image (product screenshot vs. illustration vs. video)
- Pricing display (monthly vs. annual)
- Social proof placement

### Performance
- Page load <3 seconds (Google's threshold)
- Optimize images (WebP format, lazy loading)
- Minimize JavaScript
- Use CDN for assets

## Tools for Building Landing Pages

**No-Code:**
- **Carrd** ($19/year) - Simple, fast, cheap
- **Webflow** ($0-$42/month) - More control, steeper learning curve
- **Framer** ($5-20/month) - Design-focused, animations

**Low-Code:**
- **Next.js + Tailwind** - Full control, requires coding
- **Wordpress + Elementor** - Familiar, lots of plugins

**With A/B Testing:**
- **Unbounce** ($90-225/month) - Built-in A/B testing
- **Instapage** ($199+/month) - Enterprise-grade

**Recommendation:**
Start with **Carrd** ($19/year). Validate first. Upgrade to **Webflow** or **Next.js** when you have revenue.

## Conversion Tracking Setup

**Essential Tracking:**

1. **Google Analytics 4**
   - Track page views
   - Form submissions
   - Button clicks
   - Time on page

2. **Hotjar or Microsoft Clarity**
   - Heatmaps (where people click)
   - Session recordings (watch user behavior)
   - Identify friction points

3. **Goal Tracking**
   - Form submission = Goal 1
   - Demo request = Goal 2
   - Free trial signup = Goal 3

4. **A/B Testing**
   - Google Optimize (free)
   - VWO ($186-749/month)
   - Optimizely (enterprise)

**Metrics to Track:**

- **Traffic:** Visitors, sources, channels
- **Engagement:** Time on page, scroll depth, clicks
- **Conversion:** Form fills, CTR, trial signups
- **Drop-off:** Where people leave (fix those sections)
