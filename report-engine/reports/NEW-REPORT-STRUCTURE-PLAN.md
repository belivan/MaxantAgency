# New Visual-First Report Structure Plan
**Evidence-Based with Benchmark Integration**

## Design Principles

1. **Visual First**: Charts, graphs, and data visualization over walls of text
2. **Evidence-Based**: Every recommendation backed by benchmark data or measurements
3. **Peer Learning Tone**: "Learn from industry leaders" not "you're behind competitors"
4. **Scannable**: Busy executives should get key insights in 30 seconds
5. **Actionable**: Clear priorities with effort estimates and impact projections

---

## Current Report Structure (OLD)

```
1. Executive Dashboard
   - Grade circle
   - 4 dimension scores (Design, SEO, Content, Social)
   - 3 metrics (Priority, ROI, Quick Wins)

2. Strategic Assessment
   - Text paragraph summary
   - Top priority alert box
   - Business impact table (if synthesis available)

3. 30-60-90 Day Timeline
   - Bullet list of actions per phase
   - No details, just titles

4. Visual Evidence
   - Screenshot gallery

5. Footer
```
